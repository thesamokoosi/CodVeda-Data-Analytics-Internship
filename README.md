# 📊 CodVeda-Data-Analytics-Internship
# Level 1

Welcome to my repository for the **CodVeda Data Analytics Internship**! This repository contains my Python implementation for the **Level 1 (Basic)** tasks, focusing on foundational data manipulation, cleaning, and exploratory data analysis.

## 🎯 Objective
The goal of this project is to demonstrate proficiency in handling raw data, identifying patterns, and visualizing relationships using Python. 

**Tasks Selected:**
- **Task 1:** Data Cleaning & Preprocessing
- **Task 2:** Exploratory Data Analysis (EDA)
- **Task 3 (Partial):** Generating and exporting visual reports

## 🗂️ Dataset Used
- **Dataset:** The classic **Iris Dataset**.
- **Description:** A multivariate dataset containing 150 records of Iris flowers, detailing their sepal length, sepal width, petal length, petal width, and species (Setosa, Versicolor, Virginica).

## 🛠️ Tools & Technologies
- **Language:** Python 3
- **Environment:** Google Colab / Jupyter Notebook
- **Libraries Used:** 
  - `pandas` (Data manipulation and cleaning)
  - `numpy` (Numerical computations)
  - `matplotlib` & `seaborn` (Data visualization)

## 🚀 What Was Accomplished
### 1. Data Cleaning & Preprocessing
- Inspected the dataset for structural integrity.
- Wrote algorithms to detect and handle **missing values** (via mean imputation).
- Identified and successfully removed **duplicate rows** to ensure data accuracy.
- Standardized the text formatting in the categorical `species` column to Title Case.

### 2. Exploratory Data Analysis (EDA)
- Calculated key **summary statistics** (Mean, Median, Mode, Standard Deviation).
- Visualized numerical distributions using **Histograms** (Sepal Length).
- Analyzed categorical distributions using **Boxplots** (Petal Length across different species).
- Plotted a **Correlation Heatmap** to identify the relationships between numerical variables.
- Designed a **Scatter Plot** to visualize the clustering of species based on Sepal and Petal dimensions.

## 📂 Project Structure
- `CodVeda_Level_1_Tasks.ipynb` : The main Jupyter Notebook containing all the Python code, analysis, and markdown documentation.
- `iris.csv` : The raw dataset used for the analysis.
- `iris_correlation_heatmap.png` : An exported visual plot generated during the EDA phase.
